# Getting Started  

1. Create an environment and install dependencies  
```bash
python install.py
```  

2. Activate the environment <br /> <br />
   On Linux/MacOS:
   ```bash
   source .venv/bin/activate
   ```
   On Windows:
   ```bash
   .\.venv\Scripts\activate
   ```  
3. Run the project  
```bash
python app.py
```
